# NexaAI Chatbot 🚀

Production-grade Flutter AI chatbot powered by Gemini API.

## Features
- Gemini AI Integration
- Riverpod State Management
- Clean Architecture
- Dio Networking
- Environment Variable Security
- Responsive Chat UI
- Dark Theme

## Tech Stack
- Flutter
- Riverpod
- Dio
- Gemini API
- flutter_dotenv

## Architecture
Feature-first clean architecture with:
- data layer
- presentation layer
- services layer
- provider-based state flow

## Folder Structure

```bash
lib/
├── core/
├── features/
├── app.dart
└── main.dart
```

## Setup

```bash
flutter pub get
flutter run
```

## Environment Variables

Create `.env`

```env
GEMINI_API_KEY=YOUR_API_KEY
```
